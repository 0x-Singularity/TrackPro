About Track Pro

Track Pro is a web-based project management tool designed to streamline project tracking, manage requirements, and monitor team contributions across multiple projects.

Features

  • Manage multiple projects simultaneously
  • Track functional and non-functional requirements
  • Monitor team contributions with time-tracking
  • View real-time progress reports

Purpose

Track Pro aims to consolidate project management information in one place, allowing project managers and teams to stay organized, monitor progress, and ensure project goals are met efficiently.

Stakeholders

Track Pro is sponsored by Kennesaw State University. The development team includes project manager Garrett McGuire and team members Eben Nduwimana, Zach Malta, Ben Pierce, Hong Nguyen, and Huy Nguyen.
